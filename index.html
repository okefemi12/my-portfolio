Hybrid E-Commerce Recommender

End-to-End ML System Reinforcement Learning
Live Demo Source

A hybrid recommendation system designed to model the full user journey from discovery to purchase. The system combines representation learning and reinforcement learning to optimize long-term user engagement rather than short-term clicks.

Key Architecture

  • Discovery: Variational Autoencoder (VAE) for latent preference mining
  • Intent: Transformer (SASRec) for next-item prediction (100% catalog coverage)
  • Decision: Custom RL Agent (Transformer-DQN) trained via Sim-to-Real workflows

Impact

  • Achieved 8.72% Recall@10 (1.04x lift vs. Baseline)
  • Low-latency inference (<10ms) through model optimization and efficient serving

Tech Stack: TensorFlow, FastAPI, Docker, MLflow, GitHub Actions

Credit-Scout Risk Engine

FinTech AI Explainability (XAI) LLM
Live Demo Source

A fraud detection system designed to improve model transparency by combining neural networks with explainability techniques and natural language reporting.

Key Architecture

  • Detection: LSTM Neural Network trained on PaySim financial datasets
  • Explainability: SHAP (Shapley Additive exPlanations) to visualize risk factors
  • Compliance: Llama 3.3 (via Groq) generates natural language "Adverse Action Notices" for non-technical auditors

Impact

  • Automates compliance documentation, reducing manual audit time
  • Provides granular, feature-level risk scoring for loan officers

Tech Stack: LSTM, Streamlit, Groq API, SHAP, Docker
